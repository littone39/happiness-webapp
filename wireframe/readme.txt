WIREFRAME: homepage.pdf This represents the top of the site when you first enter. It gives brief instructions
for how to use it, what the data represents, and where it came from. Users can click on a country
displayed on the map below and data for that country will be pop up in a box in a section below the map.
The nav bar at the top follows the user as they scroll through the site and allows the user to jump to 
different points on the page (namely the map and the graph).

WIREFRAME: graph.pdf This represents the page when you scroll past the map and displayed in-depth information about the 
selected countries. Users can select two variables from the drop down menu and the graph will plot each 
country. A line of best fit will also show up. Users can add multiple graphs using the plus button on the
right of the page.
